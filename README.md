原项目README：
# PythonShootGame

A simple shoot game written in Python.一个用Python编写的简单射击游戏。

# Introduction

This project only include two simple .py files: 

1. mainGame.py: The initialization and main loop of the game.
2. gameRole.py: Class of the game role.

# Requirement

1. Python 2.7
2. Python-Pygame
  
# How To Start Game
  
```bash
$ python mainGame.py
```
